# UI 設計

## デザイン方針

Google Keep のライトモード UI をベースとする。

### カラーパレット（参考）

| 用途 | 色 |
|------|-----|
| 背景 | `#FFFFFF` / `#F8F9FA` |
| サイドバー | `#FFFFFF` |
| カード背景 | 色プリセット（Keep 風パステル） |
| テキスト | `#202124` |
| サブテキスト | `#5F6368` |
| アクセント | `#1A73E8` |
| ボーダー | `#DADCE0` |

### タイポグラフィ

- フォント: システムフォント（Google Sans 相当）
- カードタイトル: 16px, font-medium
- 本文: 14px
- サイドバー: 14px

## コンポーネント

### RecipeCard（一覧カード）

```
┌─────────────────────────┐
│ [img][img][img]         │  ← サムネイル（複数並列）
│                         │
│ カレーライス              │  ← タイトル
│ 玉ねぎ、にんじん...       │  ← 材料プレビュー（省略）
│                         │
│ [和食] [定番]            │  ← タグチップ
└─────────────────────────┘
```

- カード背景色: レシピの `color` プリセット
- ホバー: 影を強調
- クリック: 詳細モーダルを開く

### InlineCreateForm（Keep 風作成）

- 初期: 「タイトルを入力...」の1行
- クリック/フォーカス: 展開して全フィールド表示
- フィールド: タイトル、材料、作り方、その他、タグ、色、画像
- 保存 / キャンセル ボタン

### RecipeModal（詳細）

```
┌─────────────────────────────────┐
│  カレーライス            [編集][×]│
│─────────────────────────────────│
│  [画像ギャラリー]                 │
│                                 │
│  材料                           │
│  玉ねぎ 1個...                   │
│                                 │
│  作り方                          │
│  1. 野菜を切る...                │
│                                 │
│  その他                          │
│  我が家の定番                    │
│                                 │
│  [和食] [定番]                   │
│                                 │
│  作成: 2026/06/01  更新: 2026/06/07│
└─────────────────────────────────┘
```

### Sidebar（タグ一覧）

```
タグ
────────────────
🟡 和食          (12)
🔵 洋食          (8)
🟢 デザート       (5)
...
```

- クリック: タグで絞り込み（複数選択 = AND）
- 選択中: 背景ハイライト
- 「すべて」リンク: 絞り込み解除

### SearchBar

- Keep 風の検索入力
- リアルタイム or Enter で検索
- クリアボタン

## アクセシビリティ

- すべてのインタラクティブ要素にフォーカスリング
- モーダル: `role="dialog"`, `aria-modal`, フォーカストラップ
- フォーム: `<label>` または `aria-label`
- 画像: `alt` テキスト（レシピタイトルベース）
- キーボード: Esc でモーダル閉じる、Tab ナビゲーション
